1. Инструкция по запуску проекта локально
Бэкенд

Перейти в папку backend:

cd coffee-site-backend


Установить зависимости:

npm install


Создать файл .env в корне backend:

PORT=5000
JWT_SECRET=yourSecretKey


Запустить сервер:

npm run dev


Сервер будет доступен по адресу: http://localhost:5000/

Примечание:

База данных реализована через JSON-файлы (data/users.json, data/products.json, data/orders.json).

Установка PostgreSQL/MySQL не требуется.

Фронтенд

Перейти в папку frontend:

cd coffee-site-frontend


Установить зависимости:

npm install


Запустить проект:

npm start


Сайт будет доступен по адресу: http://localhost:3000/

2. Реализованный функционал
Бэкенд

Регистрация и авторизация пользователей с JWT.

Система ролей (user/admin).

CRUD для основных сущностей:

Товары: просмотр, добавление, редактирование, удаление (только для админа).

Заказы: просмотр и изменение статуса (только для админа).

Пользователи: просмотр и удаление (только для админа).

Эндпоинты для админских функций: аналитика, управление контентом.

Фронтенд

Страницы: Home, Login, Register, Products, Admin.

Формы с базовой валидацией (логин, регистрация, добавление товара).

Взаимодействие с бэкенд-API через axios.

Навигация между страницами через react-router-dom.

Админ-панель с ограничением доступа по роли.

Отображение аналитики: количество пользователей и заказов.
